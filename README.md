Shows Dashboard — ABN AMRO Frontend Assignment
---

## Tech Stack

I used **Vue** + **TypeScript** (as it's the standard at ABN AMRO), **Vue Router**, **Tailwind**, **Vite**, and **Vitest**. I also chose to use **Pinia** for domain object stores, of course the Composition API alone would be sufficient for that, but I was simply curious to try it out.

Disclaimer: since I've been using **React** throughout my entire professional career, maybe some patterns in this project may not be the best ones for Vue. I appreciate your understanding :)

## Requirements

- **Node.js** `v24.14.0`
- **npm** `v11.9.0`
